                                                  US CRIME DATA REPORT
                                                  
PROJECT OVERVIEW

To analyse the crimes reported using Microsoft Excel and Power BI and find the trends and draw insights

TOOLS USED

	Microsoft Excel
	Power BI

DATASET

	Source : CRIME_DATA_FROM_2020_TO_PRESENT
	Data contains columns like date reported, date occurred, time occurred, crime code, victim age, victim sex etc.

STEPS FOLLOWED

	The data was too large to handle in Excel, therefore the cleaning was performed using Power BI. The missing values were handled and columns were formatted to exhibit the correct type.
	Used DAX formulas to create calculated columns and measures.
	Dashboard was created using column charts, gauge charts, pie charts, matrix, cards etc.
	Bookmarks, buttons and slicers were added to  make the dashboard more interactive.

KEY INSIGHTS

	Highest number of crimes were reported in the year 2022.
	Less than 10% of the culprits have been arrested so far.
	More crimes tend to occur during the evening and night hours of the day.
	The place Central serves as the crime hotspot in US.
